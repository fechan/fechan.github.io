This is a list of lexical signs in MiniSign. Function morphemes are not listed here.

Parameter descriptions

Each sign may be specified for the one of the following parameters, which together make up the sign.

Handedness
Number of hands used in the sign. If not specified, the sign is single handed by default.
Handshape
Configuration of the fingers, e.g. fist, flat hand, thumbs-up, etc.
Hand orientation
Direction the fingers and palm are pointing. When not specified, any orientation is allowed.
Hand constellation
Whether the hands are near, touching, clasped, etc. Never specified in 1-handed signs. If not specified in a 2-handed sign, the hands are apart by default.
Location
Location where the sign is articulated.
Movement
The sign's movement.
Non-manuals
Articulations that do not involve the hand. If not specified, the facial expression and body posture is neutral by default.
